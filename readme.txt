== Installation ==

1. Search for **"CF7 Mate"** in **Plugins → Add New** and click Install, or upload the zip to `/wp-content/plugins/`.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Make sure [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) is installed and active.
4. Go to **CF7 Mate → Settings** in your WordPress admin to configure features.
5. Open your page builder and insert the **CF7 Mate** module, block, widget, or element.

= Requirements =

* WordPress 6.0 or higher
* PHP 7.4 or higher
* [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) plugin (installed and active)
* One supported page builder: Divi, Gutenberg, Elementor, or Bricks

== Frequently Asked Questions ==

= Is the AI form generator really free? =

Yes — 100% free. You just need an API key from your AI provider (OpenAI or compatible). Go to **CF7 Mate → Settings → Tools** to enter your key and start generating forms instantly.

= Do I need Divi to use CF7 Mate? =

No. CF7 Mate works with Divi, Gutenberg (the default WordPress editor), Elementor, and Bricks. You only need one supported builder installed.

= Does this work without a page builder? =

Yes. The Gutenberg block works in the standard WordPress block editor — no extra page builder required. You can also use the CF7 shortcode on any page.

= Does this replace Contact Form 7? =

No. CF7 Mate extends CF7. You still create and manage forms in Contact Form 7 — CF7 Mate adds visual styling, extra field types, and powerful features on top.

= Can I save form submissions to my database? =

Yes — with Pro. The **Form Entries** feature saves every submission to WordPress and lets you view, filter, and export responses as CSV.

= Does multi-step forms work with conditional logic? =

Yes (Pro). Conditional logic and multi-step forms work together — you can show/hide fields and even skip steps based on user input.

= Is it compatible with Divi 5? =

Yes. CF7 Mate has a dedicated module purpose-built for the new Divi 5 architecture, alongside full continued support for Divi 4.

= Can I schedule when forms open and close? =

Yes (Pro). The **Form Scheduling** feature lets you set start and end dates/times for any form, with a custom message shown when the form is closed.

= Will this slow down my site? =

No. CF7 Mate only loads its scripts and styles on pages that actually contain a CF7 Mate form. Pages without forms load zero extra assets.

= Can I use the free version on multiple sites? =

Yes. The free version is GPL-licensed and can be used on unlimited sites with no restrictions.
